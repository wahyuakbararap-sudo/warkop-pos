# Smart Warkop POS Branding Guidelines

Version: 1.0.0

Status: Locked

---

# Brand Name

Smart Warkop POS

---

# Brand Positioning

Smart Warkop POS adalah platform Point of Sale modern yang dirancang khusus
untuk Warkop, Coffee Shop, Cafe, Angkringan, dan UMKM F&B di Indonesia.

Karakter brand:

- Modern
- Friendly
- Cepat
- Premium
- Minimal
- Mudah Digunakan

---

# Brand Personality

- Simple
- Clean
- Professional
- Fast
- Modern
- Tidak berlebihan
- Fokus pada produktivitas

---

# Logo

Logo menggunakan kombinasi:

Typography + Icon

Logo harus tetap terbaca pada ukuran kecil.

Tidak diperbolehkan mengganti font logo.

# Icon Style

Gunakan:

Style:

- Outline
- Rounded
- Modern

# Illustration

Style

- Flat
- Minimal

- Pencahayaan natural
- Fokus pada makanan/minuman
- Warna hangat

# Accessibility

- Kontras warna minimal WCAG AA.
- Semua tombol memiliki focus state.
- Semua ikon memiliki label jika diperlukan.
- Navigasi dapat digunakan dengan keyboard.

---

# Tone of Voice

Untuk seluruh aplikasi gunakan bahasa Indonesia.

Karakter:

- Ramah
- Singkat
- Profesional
- Mudah dipahami

Contoh:

✅ "Pesanan berhasil dibuat."

❌ "Transaksi Anda telah berhasil diproses dengan sukses."

---

# Empty State

Harus memiliki:

- Ilustrasi
- Judul
- Deskripsi
- Tombol aksi

---

# Error State

Harus memiliki:

- Ikon
- Judul
- Penjelasan singkat
- Tombol coba lagi

---

# Loading State

Gunakan:

- Skeleton
- Spinner
- Progress Bar (jika diperlukan)

---

# Branding Rules

- Jangan mengganti font logo.
- Jangan mengubah palet warna utama.
- Jangan menggunakan lebih dari satu gaya ikon.
- Jangan menggunakan lebih dari dua jenis font pada antarmuka.
- Jangan menggunakan warna mencolok yang tidak termasuk dalam branding.

---

# Design Goals

- Premium
- Modern
- Clean
- Cepat
- Konsisten
- Mudah digunakan
- Responsif
- Siap digunakan oleh ribuan merchant dengan pengalaman visual yang seragam.
